# MayDay-Aviation-Accident-Investigation-Support-System
A data science project that supports aviation accident investigations through machine learning, time-series forecasting, and historical accident similarity analysis.


## Project Description

 
The project focuses on helping investigators identify similar historical accident cases, estimate investigation duration, analyze investigation trends, and gain insights from aviation accident data.

The system is designed to assist investigators by reducing the time spent searching through historical records and providing intelligent recommendations based on past investigations. By combining machine learning, time-series forecasting, and interactive analytics, MayDay seeks to improve investigation efficiency and support evidence-based decision-making.

## Problem Statement

Aviation accident investigations often require extensive analysis of historical accident reports, aircraft information, weather conditions, operational factors, and investigative findings. This process can be time-consuming and resource-intensive.

MayDay addresses this challenge by providing a platform that can identify similar historical accidents, estimate investigation duration, forecast investigation trends, and organize relevant investigative information in a single system.

## Project Objectives

* Analyze historical aviation accident investigation data.
* Predict aviation investigation duration using regression models.
* Forecast investigation duration trends using time-series analysis.
* Develop a Similar Accident Finder system to identify related historical cases.
* Build an interactive investigation support system for aviation investigators.
* Provide insights that support investigative decision-making.

## Machine Learning Tasks

### Regression

Predict the expected duration of an aviation accident investigation based on accident characteristics and operational factors.

### Time Series Forecasting

Forecast future investigation duration trends using historical investigation data.

## System Features

* Similar Accident Finder
* Investigation Duration Predictor
* Aircraft Information Center
* Investigation Timeline Dashboard
* Investigation Trend Forecasting
* Investigation Report Support
* Aviation Investigation Resources (FDR, CVR, ATC and related investigation concepts)
